# Система управления автомастерскими

## Description

**Workshops** — консольное приложение для автоматизации работы городской службы автомастерских.

**Основные возможности:**

- Аутентификация (вход по логину и паролю)
- Управление мастерскими (CRUD)
- Управление мастерами (CRUD)
- Управление автомобилями (CRUD)
- Управление заказами (с расчётом стоимости и скидок)

## 🛠️ Installation

**Требования:**
- C++ компилятор (g++)
- SQLite3
- Make

**Шаги установки:**

1. Клонировать репозиторий:
```bash
git clone https://github.com/DanikZorin/CopyBarA-2.git
cd CopyBarA-2
```

2. Убедиться, что файл базы данных `Workshops_system.db` находится в корне проекта

3. Скомпилировать проект:

```bash
make
```

4. Запустить приложение:

```bash
make run
```

## 🎮 Usage

**Запуск:**

```bash
make run
```

**Аутентификация:**

| Роль          | Логин   | Пароль  |
| ------------- | ------- | ------- |
| Мастер        | master1 | pass123 |

**Команды (вводятся после аутентификации):**

| Команда |	Описание |
| ------- | -------- |
| -help или -? | Показать все команды |
| -tables | Список всех таблиц в БД |
| -show [таблица] |	Показать содержимое таблицы |
| -sql [запрос]	| Выполнить произвольный SQL-запрос |
| -1 или -task1	| Задание 1: заказы мастерской за период |
| -2 или -task2	| Задание 2: услуги мастера |
| -3 или -task3	| Задание 3: все заказы по маркам |
| -4 или -task4	| Задание 4: статистика по мастерским |
| -5 или -task5	| Задание 5: мастерская с макс. заказами |
| -p5 | Статистика ремонтов мастерской за период |
| -p6 | Количество выполненных работ до даты |

**Примеры работы:**

```bash
# После входа в систему
Waiting for request...
-tables
Masters
Repairs
Cars
Users
Workshops
Orders

Waiting for request...
-show Masters
id - 1
name - Иванов И.И.
phone - +79991234567

Waiting for request...
-sql SELECT * FROM Orders WHERE final_cost > 1000

Waiting for request...
-p5
Введите номер автомастерской: 1
Введите дату начала: 2024-01-01
Введите дату окончания: 2024-12-31
```
