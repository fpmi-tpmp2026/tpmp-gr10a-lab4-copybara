## Project Name
CopyBarA
## Description
Проект работает с базой данных, в которой хранятся данные о пользователях, заказах и прочие данные о мастерских. Перед работой с системой пользователь должен авторизироваться в ней, введя свой логин и пароль, которые храняться в базе данных проекта. После авторизации открывается интерфейс для работы с самой системой. В которой можно делать соответствующие запросы и вводить команды (в случае возникновения трудностей возможег ввод `% -?` для получения справочной информации). Пользователь может завершить текущую сессию и закончить выполнения программы.
## Installation
* Скопировать репозиторий проекта
* Запустить `make` для сборки проекта (либо использовать `cmake -B build && cmake --build build`)
* Запустить проект либо вручную, либо при помощи вызова `make run`
