# Система управления автомастерскими

## Description

**Workshops** — консольное приложение для автоматизации работы городской службы автомастерских.

**Основные возможности:**

- Аутентификация (вход по логину и паролю)
- Управление мастерскими (CRUD)
- Управление мастерами (CRUD)
- Управление автомобилями (CRUD)
- Управление заказами (с расчётом стоимости и скидок)

## 🛠️ Installation

**Требования:**

- C компилятор
- SQLite3
- Git
- Make

**Шаги установки:**

1. Клонировать репозиторий с кодом

```bash
git clone https://github.com/DanikZorin/CopyBarA-2.git
cd automaster-code
```

2. Создать базу данных SQLite

```bash
sqlite3 automaster.db < database/Workshop_system.sql
```

3. Скомпилировать проект

```bash
make
```

4. Запустить приложение

```bash
./automaster
```

## 🎮 Usage

**Запуск:**

```bash
./automaster
```

**Аутентификация:**

| Роль          | Логин   | Пароль  |
| ------------- | ------- | ------- |
| Администратор | admin   | admin   |
| Мастер        | master1 | pass123 |

**Главное меню:**

1.  Управление мастерскими
2.  Управление мастерами
3.  Управление автомобилями
4.  Управление заказами
5.  Отчёты и статистика
6.  Выход

**Пример отчёта по мастерской:**

Введите номер мастерской: 1
Введите начальную дату (ГГГГ-ММ-ДД): 2024-01-01
Введите конечную дату (ГГГГ-ММ-ДД): 2024-12-31

Мастер: Сергеев А.Н. | Вид ремонта: замена кузова | Стоимость: 750.00 | Скидка: 0%
Мастер: Владимиров Д.И. | Вид ремонта: замена ремней | Стоимость: 150.00 | Скидка: 0%

Итого по мастерской: 2 ремонта, общая выручка: 900.00 руб.
